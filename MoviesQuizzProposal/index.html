Movies Quizz Proposal

Description:

This is a quizz type game which pulls data from the http://www.omdbapi.com/ website (a movies database api much like imdb). The game will initially provide 1 or 2 game modes.
The first is "Answer the question", which is the usual quizz. The second type is "Timeline", in which you're trying to put the movies in the correct chronologycal order.
The game will support web based users and scoring systems (most likely via the Telerik Backend Services).

Game types explained:

Answer the question:

The player faces one of several types of questions about a single movie. The types might be:
  • "Which of the following is on the cast?"
  • "Which of the following is NOT on the cast?"
  • "Who is the director?"
  • "Guess the IMDB rating?"
  • "Guess the year?"

Example:

Jurassic Park

    Which of the following is on the cast:
  • Harrison Ford
  • Sigourney Weaver
  • Sam Neill
  • Angelina Jolie

Timeline:

Similar to the board game "Timeline". There is one random movie in the middle of the screen with its name and release year.
The player receives a random movie and tries to place it chronologically before or after the one already on the screen.
If unsuccessful, the player looses points and receives a new movie.
If succsessful, the second movie is added to the timeline, the player scores points and receives another movie. The game gets harder with every step, since there are more and more movies to consider.

Example:

  • Step one
    The player has "Die Hard" from 1988 on the screen
    Die Hard

    1988


  • Step two
    The player receives "Pretty Woman" and tries to guess was it released before or after Die Hard
    Pretty Woman

  • Step three
    The player makes a successfull guess - "Pretty Woman" was released after Die Hard, and now the board looks like this: (notice that he is now able to see the year of release of Pretty Woman)
    Die Hard

    1988
    Pretty Woman

    1900

  • Step four
    The player now gets "Batman" It is now harder to guess :)
    Batman
